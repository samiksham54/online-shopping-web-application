# Online Shopping Web Application

## Description

An Online Shopping Web Application developed using PHP, MySQL, HTML, CSS, and JavaScript.

## Features

- User Registration & Login
- Product Search
- Category Filter
- Shopping Cart
- Checkout
- Order Placement
- Product Management
- Responsive UI

## Technologies

- PHP
- MySQL
- HTML
- CSS
- JavaScript

## Database Setup

1. Open phpMyAdmin.
2. Create a new database named `wt_project`.
3. Click the **Import** tab.
4. Select `database.sql`.
5. Click **Go**.
6. Run the project using XAMPP.
